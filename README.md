# EL-FAUNO-BLANCO-SHIFTS

Aplicación React/Vite para gestionar turnos de El Fauno Blanco.

## Cómo probar en local

```bash
npm install
npm run dev
```

## Cómo generar la versión final

```bash
npm run build
```

## GitHub Pages

Este proyecto ya incluye:

- `vite.config.js` con `base: "/EL-FAUNO-BLANCO-SHIFTS/"`
- workflow de despliegue en `.github/workflows/deploy.yml`

En GitHub debes ir a:

**Settings → Pages → Source → GitHub Actions**

Después, cada vez que subas cambios a `main`, GitHub generará la web automáticamente.

URL final:

https://laulautonton.github.io/EL-FAUNO-BLANCO-SHIFTS/
